把求和符号用出花来

这是人人都会的求和符号用法，高一便学过的用于数列求和的表达方法：

∑i=mnai=am+am+1+⋯+an\begin{aligned} \sum_{i=m}^na_i=a_m+a_{m+1}+\dots+a_n \end{aligned}i=m∑n​ai​=am​+am+1​+⋯+an​​

但有时候这么写并不方便。比如如果我们需要遍历一个无序的集合：

SSS 为所有小于 100 的质数的集合。求 SSS 的元素之和。

除了把 SSS 强行规定成一个数列，还可以用这种方法写求和：

∑x∈Sx=x1+x2+⋯+xn,S={x1,x2,…,xn}\begin{aligned} \sum_{x\in S}x=x_1+x_2+\dots+x_n,S=\{x_1,x_2,\dots,x_n\} \end{aligned}x∈S∑​x=x1​+x2​+⋯+xn​,S={x1​,x2​,…,xn​}​

集合下标可以是一个判断式（如属于、不等式、等式），只要一个元素符合该判断式，则会被计入求和。

比如求一个多项式所有根的和：

∑p(x)=0x\begin{aligned} \sum_{p(x)=0}x \end{aligned}p(x)=0∑​x​

求 nnn 所有因数的和：

∑k∣nk\begin{aligned} \sum_{k\mathrel{|}n}k \end{aligned}k∣n∑​k​

我们也可以用不等式的方法求和。比如经典的 Sigma 用法就可以等价为一个不等式：

∑i=mnf(i)=∑m≤i≤nf(i)\begin{aligned} \sum_{i=m}^nf(i)=\sum_{m\le i\le n}f(i) \end{aligned}i=m∑n​f(i)=m≤i≤n∑​f(i)​

不等式中一般默认循环变量（iii, jjj, kkk等）为整数，不过也可以特别说明。

用不等式做循环的好处是可以添加多个变量，比如：

∑1≤i,j≤nf(i,j)=f(1,1)+f(1,2)+⋯+f(1,n)+f(2,1)+f(2,2)+⋯+f(2,n)+⋯+f(n,1)+f(n,2)+⋯+f(n,n)=∑i=1n∑j=1nf(i,j)\begin{aligned} \sum_{1\le i,j\le n}f(i,j)&=f(1,1)+f(1,2)+\dots+f(1,n)+f(2,1)+f(2,2)+\dots+f(2,n)+\dots+f(n,1)+f(n,2)+\dots+f(n,n)\\ &=\sum_{i=1}^n\sum_{j=1}^nf(i,j) \end{aligned}1≤i,j≤n∑​f(i,j)​=f(1,1)+f(1,2)+⋯+f(1,n)+f(2,1)+f(2,2)+⋯+f(2,n)+⋯+f(n,1)+f(n,2)+⋯+f(n,n)=i=1∑n​j=1∑n​f(i,j)​

如果两个变量的遍历规则是互相独立的，则可以把它们拆分成两次求和——比如下面的求和，iii, jjj, kkk 就是互相依赖的：

∑1≤i<j≤n0<k<2j>f(i,j,k)\begin{aligned} \sum_{1\le i< j\le n\atop 0< k< 2j>}f(i,j,k) \end{aligned}0<k<2j>1≤i<j≤n​∑​f(i,j,k)​

它等效于这样的代码：

for (int i = 1; i <= n; i++) {  for (int j = i + 1; j <= n; j++) {    for (int k = 1; k < 2 * j; k++) {      sum += f(i, j, k);    }  }}

那么如果想解出一个通项或值，就需要把三个变量分离。这往往借助于对 f(i,j,k)f(i,j,k)f(i,j,k) 做一些变形，这是竞赛中常用的手段，因为我们并不擅长在求和时同时循环两个变量。

除此以外，还有特殊的求和方法，比如轮换求和：

∑cycf(a1,a2,…,an)=f(a1,a2,…,an)+f(a2,a3,…,an,a1)+⋯+f(an,a1,a2,…,an−1)\begin{aligned} \sum_{\text{cyc}}f(a_1,a_2,\dots,a_n)=f(a_1,a_2,\dots,a_n)+f(a_2,a_3,\dots,a_n,a_1)+\dots+f(a_n,a_1,a_2,\dots,a_{n-1}) \end{aligned}cyc∑​f(a1​,a2​,…,an​)=f(a1​,a2​,…,an​)+f(a2​,a3​,…,an​,a1​)+⋯+f(an​,a1​,a2​,…,an−1​)​

穷举所有对 a1a_1a1​, a2a_2a2​, ..., ana_nan​ 的轮换（每次把第一个元素放到最后去），然后算出它们对应的 fff 的和。一共有 nnn 个这样的项。

也可以把需要轮换的变量写在下标中：

∑a1,a2,a3f(a1,a2,a3,a4)=f(a1,a2,a3,a4)+f(a2,a3,a1,a4)+f(a3,a1,a2,a4)\begin{aligned} \sum_{a_1,a_2,a_3}f(a_1,a_2,a_3,a_4)=f(a_1,a_2,a_3,a_4)+f(a_2,a_3,a_1,a_4)+f(a_3,a_1,a_2,a_4) \end{aligned}a1​,a2​,a3​∑​f(a1​,a2​,a3​,a4​)=f(a1​,a2​,a3​,a4​)+f(a2​,a3​,a1​,a4​)+f(a3​,a1​,a2​,a4​)​

对称求和：

∑symf(a1,a2,…,an)\begin{aligned} \sum_{\text{sym}}f(a_1,a_2,\dots,a_n) \end{aligned}sym∑​f(a1​,a2​,…,an​)​

穷举所有 a1a_1a1​, a2a_2a2​, …\dots…, ana_nan​ 的排列，然后算出它们对应的 fff 的和。一共会有 n!n!n! 个这样的项，因此一般只讨论两个或三个变量的对称和。

用轮换、对称求和可以非常方便地表示出一些轮换式，比如：

(i+j+k)3=∑i,j,ki3+3∑symi,j,ki2j+6ijk\begin{aligned} (i+j+k)^3=\sum_{i,j,k}i^3+3\sum_{\text{sym}\atop i,j,k}i^2j+6ijk \end{aligned}(i+j+k)3=i,j,k∑​i3+3i,j,ksym​∑​i2j+6ijk​

有时候也会在不引发歧义的情况下把下标 iii, jjj, kkk 省去。

∑i=1nai2∑i=1nbi2−(∑i=1naibi)2=∑1≤i<j≤n(aibj−ajbi)2LHS=∑1≤i,j≤nai2bj2−∑i=1nai2bi2−∑1≤i<j≤n2aibiajbj=∑1≤i≠j≤nai2bj2−∑1≤i<j≤n2aibiajbj=∑1≤i<j≤n(ai2bj2+aj2bi2−2aibiajbj)=RHS\begin{aligned} \sum_{i=1}^na_i^2\sum_{i=1}^nb_i^2-\left(\sum_{i=1}^na_ib_i\right)^2=\sum_{1\le i< j\le n}(a_ib_j-a_jb_i)^2\\ \begin{aligned} \text{LHS}&=\sum_{1\le i,j\le n}a_i^2b_j^2-\sum_{i=1}^na_i^2b_i^2-\sum_{1\le i< j\le n}2a_ib_ia_jb_j\\ &=\sum_{1\le i\neq j\le n}a_i^2b_j^2-\sum_{1\le i< j\le n}2a_ib_ia_jb_j\\ &=\sum_{1\le i< j\le n}(a_i^2b_j^2+a_j^2b_i^2-2a_ib_ia_jb_j)=\text{RHS} \end{aligned} \end{aligned}i=1∑n​ai2​i=1∑n​bi2​−(i=1∑n​ai​bi​)2=1≤i<j≤n∑​(ai​bj​−aj​bi​)2LHS​=1≤i,j≤n∑​ai2​bj2​−i=1∑n​ai2​bi2​−1≤i<j≤n∑​2ai​bi​aj​bj​=1≤i=j≤n∑​ai2​bj2​−1≤i<j≤n∑​2ai​bi​aj​bj​=1≤i<j≤n∑​(ai2​bj2​+aj2​bi2​−2ai​bi​aj​bj​)=RHS​​
